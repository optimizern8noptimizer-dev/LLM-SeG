Совместимость

Шлюз реализует OpenAI-совместимый API.
Вы можете использовать любую библиотеку или инструмент, поддерживающий OpenAI API:
openai (Python/Node.js), LangChain, LiteLLM, Continue.dev, Open WebUI и другие.

from openai import OpenAI client = OpenAI( base_url="", api_key="sk-gw-ваш_ключ" ) resp = client.chat.completions.create( model="gpt-4o", # виртуальная модель из маршрутов messages=[{"role": "user", "content": "Hello!"}] ) print(resp.choices[0].message.content)
